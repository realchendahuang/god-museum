# 馆藏状态说明

God-Museum 网站从仓库中的 Markdown 动态生成目录，不在本文档手工维护易过期的总数。

## 计数边界

- 带稳定 `id` 的文档计入正式馆藏。
- 分区 `README.md`、宪章、编目规范与书目等无稳定 `id` 文档归入“导览与规范”，不计入馆藏总数。
- `_template.md` 是写作模板，不进入网站内容集。
- `08_myths` 保存有历史来源的传统事件；`10_chronicles` 只保存 God-Museum 原创正典。

## 网站展示

完整目录会实时显示正式馆藏与导览数量，并可按类型、文明和来源状态过滤。首页的神座区域是跨文明馆藏选录，不代表全部馆藏。
