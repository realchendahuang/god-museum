# 安卡

> 安卡首先是可以书写“生命／活着”的埃及语符号，也可以成为护符和神手中的物件；“生命钥匙”是后世通行称呼，不是所有古代语境的原名。

## 物件档案

| 项目 | 内容 |
| --- | --- |
| 埃及学转写 | ꜥnḫ；对应“生命、活着”等词 |
| 符号编号 | Gardiner S34 |
| 使用形态 | 文字、护符、模具、浮雕与绘画中由神或王持有的物件 |
| 关系性质 | 神可把安卡递近王或死者鼻端，表达给予生命气息；不是某一位神的私有装备 |
| 现代误读 | “尼罗河钥匙”“万能复活钥匙”、固定男女生殖结构等说法需另证 |

## 语言与器物层

在埃及语中，安卡形符号能够书写 ꜥnḫ“生命／活着”。文字意义是理解图像的起点，却不能自动确定每一件护符如何使用。

纽约大都会艺术博物馆藏第十八王朝安卡护符模具 11.215.711，出自底比斯 Malqata；这表明安卡可被批量制作成小型护符。另一件蓝色彩釉陶护符 41.160.136 描绘一位身份写作“女神／王后”的人物，手持安卡与连枷。博物馆标题本身保留人物识别不确定性，档案也应保留。

神在图像中把安卡靠近王或死者鼻端，常被解释为给予生命或生命气息。此关系依具体铭文和场景成立，不应把安卡想象成打开杜阿特物理大门的钥匙。

## 关系

| 对象 | 类型 | 说明 |
| --- | --- | --- |
| 埃及诸神 | bearer-of | 多位神可持有或给予安卡，不能归入单一神座 |
| [法老](../../04_heroes/egyptian/pharaoh.md) | recipient / bearer-of | 接受神赐生命或在王权图像中持有，依场景变化 |
| 死者 | protective / vivifying use | 护符和葬仪图像中的生命持续，不等于取消死亡 |

## 设定层

安卡在圣物宝库不打开门，只确认一条关系仍具生命效力。没有施予者、接受者、铭文和场景时，它保持为文字与器物，不自动释放“复活能力”。

现代流行文化赋予安卡的哥特、神秘学与身份符号意义会进入接受史展柜，与古埃及层分开陈列。
