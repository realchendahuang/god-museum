# 命运泥板

> 命运泥板把命令、持有与宇宙秩序联系起来；它不是记录每个人未来的通用预言书，也不是苏美尔 me 的另一种叫法。

## 物件档案

| 项目 | 内容 |
| --- | --- |
| 阿卡德语 | ṭuppi šīmāti，复数“命运／决定”的泥板 |
| 核心作品 | [《安祖史诗》](../../08_myths/mesopotamian/anzu-and-tablet.md)、[《埃努玛·埃利什》](../../08_myths/mesopotamian/enuma-elish.md) |
| 核心功能 | 使持有者的命令获得宇宙效力、标示神议主权 |
| 主要持有者 | 恩利尔、安祖、金古，以及战后夺回者；归属按版本 |
| 物质边界 | 文学中的权力物；不能指认任意出土泥板为实物本体 |

## 两组主要叙事

在《安祖史诗》中，安祖趁恩利尔沐浴夺走泥板，导致命令与祭祀秩序停顿。他能命令武器退回原材料，显示泥板作用不是阅读未来，而是让言令生效。宁乌尔塔获胜后将其夺回。

《埃努玛·埃利什》中，提亚马特把泥板交给金古，使其命令获得权力；马尔杜克击败提亚马特阵营后夺走并控制它。两篇作品的物件是否被古代听众理解为完全同一件，需要具体文本关系研究，本馆标为 same-motif / probable-identification，不强制一条连续流转史。

## 与 me、书写和法典的区别

- [me](./me.md)是文明职能与神圣范畴集合，不是一块命运泥板；
- [纳布](../../02_deities/mesopotamian/nabu.md)记录命运，不等于所有版本中持有本泥板；
- 历史法典石碑、占卜泥板和行政账本不是命运泥板的考古复制品；
- “谁拿到就全知全能”是现代奇幻规则，不由史诗支持。

## 设定层

宝库不保存一块脱离文本的唯一命运泥板，而保存两块带版本标签的权力投影。只有进入对应神话剧场时，它们才恢复命令效力。

任何神试图把它变成跨神系统治工具，泥板都会先要求出示授予命令权的那场神议。
