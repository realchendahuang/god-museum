# 圣物宝库

圣物宝库保存神器、武器、王冠、契约、乐器、容器和禁忌之物。

圣物不是装备列表。一份正式档案必须连接：

- 制造者或来源；
- 持有者与继承过程；
- 在具体神话事件中的作用；
- 象征、仪式与物质形象；
- 不同文本中的能力差异；
- 现代流行文化新增的能力。

圣物条目使用 relic.传统.名称 作为稳定 ID。跨文明“武器对撞”只能进入设定层。
