# 来源目录

来源目录为所有条目提供稳定引用节点。

## 来源等级

- **P1 · 原始材料：** 同时代或传统内部的文本、铭文、仪式、器物与图像。
- **P2 · 前现代整理：** 注疏、神谱、史书和较晚但具有传统影响力的整理。
- **S1 · 专门研究：** 学术专著、论文、校注与研究型译本。
- **S2 · 参考工具：** 学术百科、博物馆目录和可靠数据库。
- **R · 接收史材料：** 小说、影视、游戏和现代艺术，只能证明后世形象。

等级表示材料用途，不表示“越早越真实”。较晚材料可以直接证明较晚传统，但不能无说明地证明更早时期。

## 使用规则

- 正文引用稳定 source ID；
- 完整书目信息进入 [bibliography.md](./bibliography.md)；
- 网络地址只作为访问路径，引用仍需写明作品和具体定位；
- 译文必须标明译者或说明为项目自译；
- 搜索摘要、百科转述和 AI 生成内容不得成为正式来源节点。
