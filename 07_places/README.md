# 神域与地点

这里保存奥林匹斯、九界、天庭、昆仑、杜阿特、冥府、神山、圣河、城邦和祭祀地点。

## 两类地点

### 神话地点

存在于神话叙事和宇宙结构中的地点。它们服从本土传统，不被强制拼成一张统一地图。

### 人间地点

可以定位的山川、城市、神庙、遗址和祭祀中心。它们是神座进入人间的锚点，也可能构成文明通道。

## 编目要求

- 不把同名神话地点与现实地点自动等同；
- 不用现代国界覆盖古代文化区域；
- 记录不同文本对地点位置与功能的分歧；
- 众神殿与诸神域的原创空间明确进入设定层。

地点条目使用 place.mythic.传统.名称 或 place.historical.传统.名称 作为稳定 ID。

## 已开放地点

- [瓦尔哈拉](./valhalla.md)：与奥丁和阵亡者相关的北欧神话地点，不是所有死者的统一天堂。
- [Þrúðvangr 与 Bilskírnir](./thrudvangr-bilskirnir.md)：索尔的领域与大厅，让守护者拥有可返回的秩序。
- [众神殿](./pantheon.md)：项目自身的设定空间，说明共殿如何出现、运作、变化并限制权力。
- [昆仑／昆仑之丘](./chinese/kunlun.md)：帝之下都、河源、西王母周边与后世仙境重叠但不完全同图的多版本神山。
- [不周山](./chinese/buzhou-mountain.md)：西北“不合”之山与共工撞断天柱的宇宙结构节点。
- [扶桑与汤谷](./chinese/fusang-tanggu.md)：十日沐浴、九居下枝一居上枝的东方太阳地理。
- [蓬莱与海上五山](./chinese/penglai-five-isles.md)：归墟、巨鳌承山、两山沉没和仙圣迁徙构成的海上系统。
- [幽都之山](./chinese/youdu.md)：黑水、玄色动物与异民组成的北方地理，不直接等同后世官僚化地府。
- [泰山](./chinese/mount-tai.md)：真实山岳、秦汉封禅与后世东岳／治鬼面相按制度和年代分层。
- [河／黄河神圣水系](./chinese/yellow-river-hebo.md)：现实水系、河伯诗歌空间、治水与国家山川祀典叠合但不互相替代。
- [社稷坛与社坛](./chinese/sheji-altars.md)：由王、诸侯与群体分别建立的土地祭坛体系，不是一座全国唯一神殿。
- [鹤鸣山／鹄鸣山](./chinese/heming-mountain.md)：张陵学道的史书地名、老君授道圣地与现代四川地理认定分层。
- [三清境／玉清、上清、太清](./chinese/sanqing-realms.md)：经典、神谱、仪式与宫观把三清空间化的历史结果，不吞并全部道教诸天。
- [杜阿特](./egyptian/duat.md)：太阳夜航、奥西里斯王国、死者转化与守门者发生作用的多版本神话空间。
- [赫利奥波利斯](./egyptian/heliopolis.md)：真实城市 Iunu 与拉—阿图姆、第一丘和太阳祭祀记忆重叠的历史地点。
- [阿拜多斯](./egyptian/abydos.md)：早期王墓、地方死者神、奥西里斯祭祀、朝圣和每日供奉叠合的历史地点。
- [尼普尔与 Ekur](./mesopotamian/nippur-ekur.md)：不依赖长期帝国首都身份的恩利尔王权中心。
- [埃里都与 E-abzu](./mesopotamian/eridu-eabzu.md)：早期城市、淡水深渊和恩基智慧的重叠地点。
- [乌鲁克与 Eanna](./mesopotamian/uruk-eanna.md)：城市化、伊南娜、安与吉尔伽美什记忆交汇之地。
- [巴比伦与 Esagila](./mesopotamian/babylon-esagila.md)：马尔杜克升王和城市宇宙中心化的历史空间。
- [博尔西帕与 Ezida](./mesopotamian/borsippa-ezida.md)：纳布书写传统与巴比伦节庆道路的另一端。
- [Irkalla](./mesopotamian/irkalla.md)：由多种语言和文本组织的两河冥界，不是一张固定地下地图。
- [乌加里特／拉斯沙姆拉](./ugaritic/ugarit-ras-shamra.md)：王宫、神庙、住宅档案和港口共同构成的真实青铜时代城市。
- [萨庞山](./ugaritic/mount-saphon.md)：巴力宫殿、风暴地景与 Jebel al-Aqra 之间的神话—地理关系。
- [埃尔的河源与双深水](./ugaritic/el-source-of-rivers.md)：神议高位的诗歌居所，不是已确认考古王宫。
- [莫特之域](./ugaritic/mot-underworld.md)：死亡神的吞噬空间，不吞并全部乌加里特亡灵传统。

地点档案不会把它们强行放进一张总地图。瓦尔哈拉、杜阿特与索尔神域保留各自文本关系；赫利奥波利斯和阿拜多斯则同时显示考古地景与神话定位。众神殿只在文明交汇时提供门，不吞并门后的宇宙。
