# 神域与地点

这里保存奥林匹斯、九界、天庭、昆仑、杜阿特、冥府、神山、圣河、城邦和祭祀地点。

## 两类地点

### 神话地点

存在于神话叙事和宇宙结构中的地点。它们服从本土传统，不被强制拼成一张统一地图。

### 人间地点

可以定位的山川、城市、神庙、遗址和祭祀中心。它们是神座进入人间的锚点，也可能构成文明通道。

## 编目要求

- 不把同名神话地点与现实地点自动等同；
- 不用现代国界覆盖古代文化区域；
- 记录不同文本对地点位置与功能的分歧；
- 众神殿与诸神域的原创空间明确进入设定层。

地点条目使用 place.mythic.名称 或 place.human.名称 作为稳定 ID。
