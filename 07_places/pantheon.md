# 众神殿

> 众神殿不是所有神话背后隐藏的“真正宇宙”，而是文明彼此承认、翻译和记住对方之后出现的交汇空间。

## 地点性质

众神殿同时是议庭、档案馆、圣所、记忆之城与跨文明会面地，但它不属于任何神系，也不凌驾于任何诸神域。

它只共享一套展陈和交往规则：

- 每位神保留独立神座；
- 每套神话保持自己的天地、时间与死亡秩序；
- 大殿按权柄组织入口，不制造新神系；
- 历史传播与原创关系必须显式标注；
- 来源不足时，建筑宁可保持黑暗，也不自动补完。

## 如何出现

众神殿没有单一创世神话。它的第一扇门出现在人类第一次承认异族的神也是神时。此后，翻译、贸易、迁徙、战争、考古发现和重新讲述不断增加新的道路。

一条道路能够打开，不代表双方神祇已经融合。文明接触只提供相遇条件，身份关系仍需原典和历史证据判断。

## 空间结构

| 区域 | 功能 | 不是什么 |
| --- | --- | --- |
| 十二大殿 | 按权柄进入神座 | 战力赛场或统一神系 |
| 神座 | 保存一位神的完整身份 | 可随意复制的角色卡 |
| 文明通道 | 记录真实接触与传播 | 无历史依据的传送门 |
| 英雄长廊 | 保存人神边界上的行动者 | 次级神殿 |
| 圣物宝库 | 保存制造、持有与事件关系 | 通用装备仓库 |
| 异兽庭院 | 保存非人存在及翻译边界 | 怪物图鉴 |
| 神话剧场 | 并排展示事件版本 | 宣布所有母题同源 |
| 静默长廊 | 保存未知与失传 | 任意原创的空白支票 |

## 权限与限制

众神在本土诸神域中的权威不会自动延伸到共殿。宙斯不能以奥林匹斯王权统治因陀罗，奥丁不能征召其他文明的阵亡者，阿耆尼也不能绕过邀请把任何神带入陌生祭祀。

众神殿不裁定哪套神话“真正发生过”，只裁定一条陈述是否诚实说明了自己的来源层次。

## 变化规律

- 新材料被发现时，神座会增加面相而不是覆盖旧版本；
- 一个名字失去所有人间记忆时，对应道路会暗下，但不会立刻消失；
- 真实历史中的神格融合会生成可追踪的新关系；
- 纯原创相遇可以发生，但会在空间上留下朱砂色标记；
- 当现有分类容纳不了一位神时，建筑必须改变，而不是删改神。
