# 众神殿

> 众神殿不是所有神话背后隐藏的“真正宇宙”，而是文明彼此承认、翻译、记得和争论对方之后出现的交汇空间。

## 地点性质

众神殿同时是会面地、档案馆、圣所、议事地与记忆之城。它连接诸神域，不位于诸神域之上；它拥有通行规则，没有宇宙主权。

共殿不创造神性，不裁定哪套神话最终真实，不用座次、胜负或信徒数量排列神系高低。

## 如何出现

众神殿没有唯一创世神话。它的第一扇门出现在人类第一次承认异族的神也是神时。

此后，每一次真实的贸易、迁徙、征服、翻译、宗教传播、考古发现和重新讲述，都可能留下一段道路。历史道路只证明相遇条件存在，不自动合并神座。

## 抵达

一位神进入众神殿时，到来的不是可以一次调用所有时代权力的完整角色，而是由特定面相构成的一次显身。

显身到来时，门会记录：

- 哪些面相正在发言；
- 通过历史门、礼请门、朱砂门还是裂隙到来；
- 可在殿内行使什么权力；
- 这些行动需要哪些名字、圣物、仪式、同意与地点条件；
- 何时离开，以及使用权力后必须承担什么代价。

只剩名字、图像、称号或残片的存在以回响形式进入，在身份未定前不会生成完整现代人格。

## 空间如何运行

| 区域 | 能够发生的事 | 不拥有的权力 |
| --- | --- | --- |
| 门庭 | 核对显身、同意、条件和可携带对象 | 为所有神自由开放他者诸神域 |
| 十二大殿 | 按权柄看见差异、发起有限议事 | 把权柄相似变成同一神系或战力组别 |
| 神座 | 保存持续身份、面相、关系与异议 | 为来源空白补造人格 |
| 文明通道 | 展示真实交流与身份变化 | 用现代类比伪造历史传播 |
| 英雄长廊 | 让人神边界上的行动者作证与选择 | 把功业换算为自动神座 |
| 圣物宝库 | 保存制造、持有、仪式与事件关系 | 将圣物改造为通用装备和伤害数值 |
| 异兽庭院 | 保存非人存在与它们的本土边界 | 把一切敌对存在改成可召唤战兽 |
| 神话剧场 | 并列事件版本，在边界内重演 | 把原创结局倒写进古代原典 |
| 静默长廊 | 保存眠座、残响、被吸收前史与失名位置 | 把未知当作任意原创的空白支票 |

## 议事与止损

众神殿没有常设最高法官、最高书记或最高神王。任何著名神都只能在明确受邀、具有相关经验的事件中参与。

门、身份或跨域行动发生争议时，共殿只使用有限六步程序：

1. 封门止损；
2. 各方陈述；
3. 展示来源与条件；
4. 受影响者提出异议；
5. 作出范围、期限和代价明确的处置；
6. 将少数意见与未决问题永久留档。

结论只对该门、该关系和该事件有效，不形成跨文明普遍法。

## 变化规律

- 新材料改变可显现的面相，不改写人间过去；
- 一个名字失去稳定锚链时，神座可从明座转为隐座或眠座，不按信徒数量简单死亡；
- 回响获得新证据时可形成有限面相，不会自动恢复材料从未保存的记忆；
- 真实历史融合建立可追踪关系，原创交汇留下朱砂标记；
- 一次事件的胜负不会改写任何神在本土原典中的权威；
- 当现有分类无法容纳一位神时，建筑必须改变，而不是删改神。

## 人类的位置

人类是命名者、译者、见证者、发掘者、请求者和继承或拒绝者。当争议涉及人类死者、遗物、仪式、神名或圣地时，相关人类共同体必须获得不能被神代替的陈述位置。
