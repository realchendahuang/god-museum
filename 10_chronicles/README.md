# 众神编年史

众神编年史保存 God-Museum 的原创正典。这里的事件不是任何文明失传的古代神话，也不反向证明参与者在历史中真实相遇。

每篇纪事都必须从已有馆藏和来源边界出发，让两条以上同样正当、却无法同时满足的规则发生冲突。事件的结果不是战力排名，而是一扇门、一个面相、一条关系、一项盟约或一种记忆状态发生永久变化。

## 阅读规则

- 传统神话和历史事件继续进入[神话剧场](../08_myths/README.md)；
- 原创纪事统一标记 `canon_layer: setting`；
- `source_status` 只表示参与者与来源锚点已经建档，不表示故事来自古代材料；
- 没有历史接触的会面使用礼请门或朱砂门，不伪装成文明传播；
- 人类译者、见证者、死者与继承者不能在神到场后失去陈述权；
- 每篇结尾必须明确列出代价和永久变化。

## 首批纪事

1. [《无主之雷》](./the-unclaimed-thunder.md)：一幅失去语境的雷神图像同时唤起四个神座。
2. [《死者走错的门》](./misrouted-dead.md)：错误译注把三套死后程序接到同一条亡者通道。
3. [《重新被念出的名字》](./name-spoken-again.md)：残片让一个静默名字获得有限显身，却仍不能确认完整身份。
