# 设定基石

这一目录保存 God-Museum 的不可绕过规则。任何神明、神话事件、文明通道和原创纪事都必须先满足这里的约束。

## 最高原则

- [《众神殿宪章》](./constitution.md)：世界如何成立，众神殿拥有什么权力，又明确没有什么权力。
- [《三层正典》](./canon.md)：原典层、整理层和设定层如何彼此隔离。

## 运行法则

- [《世界模型》](./world-model.md)：人间、诸神域、众神殿与神话剧场的四层结构。
- [《时间、面相与显身》](./time-and-manifestation.md)：四种时间，以及神座、面相、显身和回响的区分。
- [《门与文明通道》](./gates-and-corridors.md)：历史门、礼请门、朱砂门与裂隙的开启条件。
- [《权力边界与争议程序》](./jurisdiction-and-disputes.md)：跨域行动、有限裁定和异议留档。
- [《人类、记忆与复苏》](./humans-memory-and-revival.md)：人类的六种行动者角色、五种记忆状态与复苏锚链。
- [《叙事发动机》](./narrative-engine.md)：原创纪事如何从来源边界中生成冲突，并留下永久后果。

## 编目约束

- [《编目协议》](./cataloging.md)：条目 ID、元数据、状态与引用格式。
- [《关系类型》](./relation-types.md)：历史、比较、神话内部与原创设定关系的精确定义。

## 总原则

> 先保存神的来处，再让神进入众神殿。

目录结构、页面模板和分类都可以调整；“不以比较抹去身份、不以原创冒充原典”不可调整。
