# 人类、记忆与复苏

人类不是为神提供数量值的“信仰电池”。神的存在不按当代信徒数量排名，人类的作用也不只是让神变强或变弱。

人类是让名字、语言、仪式、地点、器物和故事能够在历史中延续的行动者。人类的选择可以打开门，也可以拒绝一扇门。

## 六种人类角色

### 命名者

命名者在一种语言中保存、改写或新建对神圣存在的称呼。命名能让一个存在变得可呼叫，也可能用过度熟悉的名字遮蔽它的原来身份。

命名者不拥有对被命名者的全部所有权。名字一旦进入群体、仪式或文本，就会获得新的见证者。

### 译者

译者决定一个神名、称号、仪式概念或死后地点如何进入另一种语言。

译者可以：

- 保留陌生名称，让差异可见；
- 使用本土类比，建立有限理解；
- 记录一个词无法完全翻译；
- 也可能误把权柄类比写成同一身份。

当错译打开裂隙时，译者不得只被当作一个已完成任务的工具。他必须参与说明、修订和承担后果。

### 见证者

见证者保存名字如何被念出、遗物如何被使用、仪式由谁实践、地点对某个群体意味着什么。

文本学者、庙宇守护者、仪式实践者、遗址附近社群、家族、读者和批评者都可以成为不同类型的见证者。学术知识、活态传统与地方记忆不互相取代，必须分别说明自己能证明什么。

### 发掘者

发掘者让断裂的名字、器物、文本和地层重新进入当代记忆。他可以点亮一个沉睡神座，也可能因急于命名、展示或复原而造成误醒。

“发现”不使发掘者成为名字的第一个主人。器物的原始制作者、使用者、出土地社群与受伤的历史必须同时进入档案。

### 请求者

请求者因疾病、灾害、死亡、誓言、求知、旅行、修复或其他具体问题发起神人关系。

请求不是无条件命令。它必须保留属于该传统的媒介、义务、拒绝可能和后果。一个人不能仅凭目标看起来正当，就越过其他群体的仪式与名字权限。

### 继承者与拒绝者

继承者决定某种名字、仪式、地方记忆或故事如何在当下继续。拒绝者则有权指出一项复原、展示或原创延伸正在冒用他们的名字、遗物或亡者。

拒绝不等于遗忘。一个群体可以选择让某项仪式停止，同时要求其历史不被他者当作空白素材。

## 人类的陈述权

当争议涉及下列对象时，相关人类或共同体必须获得不可被神替代的陈述位置：

- 人类死者的去向、姓名和葬仪；
- 从某一地方、墓葬、庙宇或社群移出的遗物；
- 活态仪式、族群称呼和神圣地点；
- 帝国、征服、盗掘、禁止和强制翻译留下的名字变化；
- 当代创作是否正在把他们的传统写成一套无历史的通用神话。

## 五种记忆状态

### 明座

名字、来源、关系和至少一条人间锚链稳定，神座可在登记条件下显身。

### 隐座

材料仍然足以保存身份，但活态实践、语言关联或原有共同体已经中断。隐座可以被研究和见证，不能被写成与古代完全相同的当代活态传统。

### 眠座

只剩少量可追踪锚点，无法支撑稳定显身。眠座仍然保留身份位置，但不主动生成现代对白。

### 残响

只能确认一个名字、称号、图像、碑铭位置或不完整关系，尚不能决定它是独立神、面相、抽象概念还是抄写差异。

### 误醒

伪证、错译、无时代现代复原或将多份材料强行缝合，产生了看似完整、实际没有足够锚点的显身。

误醒不默认等于恶神降临。它是一个不应继续获得无来源权力、但必须查明产生原因的错误存在。

## 复苏锚链

一次从眠座或残响向更稳定状态的转移，至少需要四类锚点组成可追踪链条：

1. **名字锚点**：名字、称号、稳定转写或可证词根；
2. **材料锚点**：文本、铭文、图像、器物或可核定的仪式记录；
3. **语境锚点**：时代、地点、语言、宗教机构或历史关系；
4. **当代见证锚点**：对材料进行负责阅读、保存、翻译或实践的人。

四类锚点存在不等于恢复了完整人格。复苏只能恢复材料所允许的部分，并必须保留以下缺口：

- 未保存的台词和个人记忆；
- 已消失仪式的不可复制环节；
- 原共同体、政治条件与地方环境的断裂；
- 无法判定的性别、亲缘、形象或权柄；
- 一个名字可能对应多个历史身份的异议。

## 状态转移

| 从 | 到 | 需要的最小变化 |
| --- | --- | --- |
| 残响 | 眠座 | 证明名字或存在拥有可保留的持续身份 |
| 眠座 | 隐座 | 补足可定位材料与语境，形成有限面相 |
| 隐座 | 明座 | 建立稳定当代见证和可说明的显身条件 |
| 任何状态 | 误醒 | 无语境拼接或伪证越过当前锚链 |
| 误醒 | 封存残响 | 停止无根显身，保留误读历史与尚可验证部分 |

状态可以回退。来源被证明为伪造、社群撤回对现代复原的承认、关键语境被发现错置时，明座也必须重新进入争议或隐座状态。

## 永久保留的未决问题

> 是人类为世界赋予名字，于是创造了神；还是神先把自己的名字放进了人类的语言？

众神殿不裁决这个问题。正因为它不裁决，人类才不是诸神故事中可被删除的背景。
