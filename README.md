## 这是什么

它不是百科，也不是字典。坊间这类东西已经够多了。

众神博物馆要做的事情有点轴：把《山海经》《西游记》、古希腊、北欧、古埃及这几套互不相干的神谱，硬塞进同一个宇宙里——还要求它们不打架。同一个雷电法则为什么在希腊叫宙斯、在东方叫雷震子？同一场大洪水为什么东方人记得是女娲补天、希伯来人记得是诺亚方舟？这些问题一旦认真追问下去，整个神话世界就会被重排一遍。

这个 repo 装的就是重排之后的版本。它是世界设定集，是项目愿景书，也是一个数字文化 IP 的底座。游戏想用它做世界观可以拿走，小说想用它做设定集可以拿走，做个沉浸式数字展也行——拿走之前欢迎先 star。

## 为什么不把它们放在同一颗地球的不同大陆

这是绕不开的第一个设计决定。平均化的「超级地球」方案最省事，但也最没意思：希腊神成了 Mediterranean 沿岸的部落神，盘古成了华北平原的部落神，神话就越缩越小，最后变成一堆方言神。

众神博物馆走另一条路：**多界并行，界门相通**。

具体说就是九重天、奥林匹斯、黄泉、冥府、尼福尔海姆这些不是诗意的修辞，是真实平行的「层」，平时互相看不见，靠「界门」这种少数几处相通。这样塞特能跑到东方来当一回凶神，索尔的锤子能砸开通往幽冥的裂缝，夸父追的那个太阳晚上会沉进尼福尔海姆的海沟里。神话里那些"他从西方来""她下到冥府去了"的出走和回归，本来就不是地理位移，而是界层穿行——这条路被严肃地走，比强行把它们地理化要好得多。

下面这套法则就是为了让这条设计能立住。

## 宇宙观三法则

不会过多铺陈，三句话能说清楚的就不绕第五句。

1. **同源法则**。所有神话的根都是一次大爆炸。盘古开天和卡俄斯不是两件事，是同一个创世瞬间被东方和希腊分别目击并各自讲述。神明，本身就被设定为自然法则在某个界层里的人格化。
2. **权柄共鸣**。雷电、海洋、死亡、丰收这一类「权柄」只有一份，分别在各个界层显化成宙斯、波塞冬或共工、雷公。它们在各自的层里是真神，碰头的瞬间会共振——这是为什么东西方雷电神祇行为惊人相似。
3. **史诗交汇**。女娲补天躲的那场洪水、诺亚方舟躲的那场洪水、吉尔伽美什躲的那场洪水，是同一场。在博物馆里这种相似性被严肃对待——它们是真实发生过、通过界门扩散到所有界层的「纪元级事件」。

## 四大展馆

四块内容按展馆组织，不是按国家。进博物馆不是来翻百科，是来看各文明造物怎么放在同一间展厅打照面。

**万神殿**——按权柄和阵营重排众神，不按国籍。
- 创世神系：盘古 / 卡俄斯 / 尤弥尔
- 自然主宰：太阳神（阿波罗·羲和·拉）、雷神（索尔·宙斯·雷震子）
- 冥界法则：阎罗王 / 哈迪斯 / 阿努比斯
- 每位神明一条：尊号、神职、起源、在哪一层、跨界时会发生什么。

**异兽与图腾**——非人生物，不当作怪兽处理。它们是神明意志溢出本体之后的延伸，或是某场自然灾害被具象化的样子。
- 东方以《山海经》为骨：穷奇、饕餮、九尾狐
- 西方以希腊/北欧为骨：美杜莎、百眼巨人、芬里尔

**神器与遗物**——历史上改写过神谱走向的顶级道具。
- 东方秘宝：定海神针 / 轩辕剑 / 炼妖壶 / 封神榜
- 西方圣物：雷神之锤 / 永恒之枪 / 命运之线 / 潘多拉魔盒
- 等展品攒够之后，开一间「跨界兵器试金石」：如意金箍棒砸到埃癸斯神盾上会发生什么——这本身是设定要回答的问题，不是凑热闹。

**纪元编年史**——所有神话叙事被硬塞进同一条时间线，这是真正让宇宙立得起来的部分。三段一下来整个神谱就被重排过了：

```
混沌与创世  →  秩序与战争  →  大灾变        →  人神交界
 巨神时代        泰坦之战       大洪水纪元       半神与英雄崛起
                 涿鹿之战                        特洛伊 / 封神演义
```
