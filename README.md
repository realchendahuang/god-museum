## 初心

我想盖一座馆，让宙斯、湿婆、盘古、奥丁能站进同一间屋子，不别扭，不打架，也不互相吞掉。

希腊人讲雷神掷矛，北欧人讲雷神抡锤，中国人讲雷公敲锤。三个故事各是各的，凭什么这么像？再往下翻还有更多：全世界都讲过大洪水，都讲过冥府去了回不来，都讲过有人为了全知赔了一只眼。

我心里一直有个画面：这些神本来就是同一拨角色，各自被不同的文明从各自的角度画了一笔，画的全是同一个宇宙。差别在讲法，宇宙是同一个。

众神博物馆就是想把这幅画拼回来看。它是一套世界观，也是一个创作场地。我在这搭一个能装下东边和西边、神和物理、神话和科幻的框架，然后让故事在框架里长出来。

---

## 这是什么

一个世界观仓库。

它要在一条底线上同时容下三方：现代科学、东方神系、西方神系。科学提供地基，神系住在地基上。神不用被科学改造，也不用被科学赦免。仓库收集的是它们能共存的那些点，把它们写成展品。

仓库里每件展品都是一次同框：一个真实现象挂在中间，周围一圈是各文明讲它的故事。读者读到的是"这个房间居然真能装下这么多"，而不是"哪个对哪个错"。

---

## 三条规矩

**地基让给科学，房子让给神。** 现成的物理定律和现象拿来做地基，越稳的越好。神住在地基上，怎么活、怎么打架、怎么老，按各文明讲的来，不改原文。地基不动神，神不拆地基。

**各家讲法各自摆着。** 希腊说宙斯被骗就让他被骗，东方说太阳是羲和生的就让她生。我们不删神话，不统一神话，不替神话打架。所有故事原样摆进展厅，让它们自己打照面。

**对不上的地方也是展品。** 神话之间、神话和科学之间，会有对不拢、接不严、半套能讲半套讲不下去的地方。这些地方不丢回炉，它们本身就是博物馆最值得停留的角落。讲解员也会挠头，挠头的内容也算藏品。

## 几件开场展品

下面这几件是仓库的开张摆位。一件展品长什么样，看完就有了手感。后面新加的，照这个手感来，不必照这个结构抄。

### 展品一 · 创世巨尸

46 亿年前一团分子云在引力下坍缩，中心点燃成太阳，外围岩屑低速冷凝成行星。你和身边所有比氢重的东西，都来自那团云后来的恒星燃烧与爆炸。

盘古开天，身化万物，左眼为日，右眼为月，血为江河。尤弥尔被兄弟肢解，肉成土，血成海，颅骨当天。印度原人普鲁沙被分尸成日月。全世界讲宇宙起点时，几乎都在讲"一个巨大存在被拆成了世界"。

把这间展房照亮的是：那团云真的就是被拆开了的庞然大物，它身上每一块最后都长成了别的东西。神没有发现星际化学，但他们画对了姿态——世界是从一具被拆开的躯体里长出来的。这具躯体后来有了名字，叫分子云。

### 展品二 · 雷神击空

云和地之间积到 3 亿伏电位差，击穿空气形成导电通道，瞬间放电。那道闪电的形状就是这条通道的形状。

索尔掷锤。宙斯抛矛。雷公敲锤。希腊把霹雳做成宙斯的招牌武器，北欧把妙尔尼尔做成索尔的手。全世界都把闪电讲成有人格的神的一击。

导通道是事物本身，锤矛是它的脸。神把脸画得很准——一击必中、瞬间、致命、来自天上。放电通道发生在哪户人家的庄稼地上，那一户就把它叫做他们那位神的一击。

### 展品三 · 太阳会老

太阳每秒聚变 6 亿吨氢。50 亿年后它会膨胀成红巨星吞掉地球，再萎缩成白矮星。它一直在变，只是变得慢，一个人一辈子看不见。古人有几十代人叠在一起的口传，反而能察觉"它在烧自己"。

拉乘太阳船，日日衰老，每晚入冥府重生平晨。阿波罗驾马车，黄昏必落。羲和生十日，后羿射下九个——十个太多了，得射下来。全世界都把太阳讲成"活的、每天在变、晚年会出乱子"。

古人没有五十亿年这个尺度，但他们感觉到太阳是个会变的东西。后羿射九日，是他们替"太阳可能越来越多越来越烧"做的一个神话版预警。神宫哪天失控，下界就烧起来——这一句古人比我先说。

### 展品四 · 长生要吃苹果

细胞每分裂一次，端粒就短一截，短没了细胞不再分裂，人就老就死。端粒酶能补，补过头就是癌。永生和癌变在生物学上是同一件事的两端——想不死，先要冒变成癌的风险。

北欧讲众神必须一直吃伊登的苹果才能保持年轻，吃一次不够，要一直补，哪天伊登断了众神就开始衰。希腊讲爱神赐凡人提托诺斯永不死，却忘给永青春，提托诺斯枯萎成一只折磨。东方讲神仙炼丹续命，断了丹就跌境。

伊登篮子里的苹果就是端粒酶，续命的活儿没有终点。所有文明都在同一个地方卡住：永生是个有期限的图纸，延长它等于跟自己的细胞周期硬刚。

### 展品五 · 全知要赔一只眼

一个系统掌握的信息越多，自身越有序，代价是宇宙其余处更无序。Shannon 把这条路走到了底，热力学第二定律守在那里。涨这边，必须塌那边。

北欧密米尔之井——奥丁喝一口涨全知，代价是赔一只眼。希腊阿波罗能预言，但总不及命运三女神。印度湿婆第三眼睁开即全知。

所有故事里"换一只眼换全知"那个桥段，是在用诗讲同一件事：你想看得更多，你自己就得塌得更狠。这条规矩压在所有神上面，连神也越不过。它叫信息守恒。神不全知，不是神不够大，是宇宙不让人那么大。

### 展品六 · 冥府回不来

一个系统熵只会变大不会变小。死者热量散尽、结构崩解，是熵增到极点的稳态。把它逆转过来要从外界吸更多能量补偿，越补偿越加速外界熵增。

希腊冥府，下去就回不来，仅极少数英雄破例。东方黄泉，黄的是地下水位。北欧赫尔，冷暗的终局之地。全世界都把冥府讲成"回不来"。

埋下去的人真的回不来，这件事每个民族都看见了。冥府是这个事实的展房。"回不来"三个字直接就是热力学第二定律——回去要违反它才能成。神能跨界，但连神也从熵极点回不来。

## 目录

仓库按上面的思路分间，不按国家分。一间展房里挤的是各文明的角色在同一桩事上打照面的样子。

```
god-museum/
├── README.md
├── README_EN.md
├── LICENSE
├── 📁 01_Protocol (协议)
│   └── Canonical_Stitching.md       # 一件展品怎么写：现象居中，故事环绕
├── 📁 02_Cosmology (宇宙观)
│   ├── Genesis.md                   # 创世巨尸
│   ├── Solar_Cycle.md               # 太阳会老
│   └── Underworld.md               # 冥府回不来
├── 📁 03_Pantheon (众神)
│   ├── Lightning.md                 # 雷神击空
│   ├── Knowledge.md                 # 全知赔眼
│   └── Immortality.md               # 长生吃苹果
├── 📁 04_Anomaly (异常收容)
│   └── Glitch_Bestiary.md           # 法则溢出、被神化的怪——暂留位
├── 📁 05_Armory (因果兵器)
│   └── Relics.md                    # 改写局部物理常数的神物——暂留位
├── 📁 06_Chronosphere (编年史)
│   ├── The_Flood.md                 # 大洪水：末次冰期海侵 vs 女娲 / 诺亚 / 吉尔伽美什
│   └── Timeline.md                  # 所有展品按时间线收束
└── 📁 99_Workshop (草稿)
    └── (还没铺开的条目先停这里)
```

展品怎么写有手感的参考，写在 `01_Protocol/Canonical_Stitching.md`。仓库是活的，分间会随着新展品变。
