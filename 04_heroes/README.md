# 英雄长廊

英雄长廊保存半神、文化英雄、圣王、勇士、先知、弑神者和进入神域的凡人。

英雄不是缩小版神明。档案必须说明：

- 出身是否真的包含神性；
- 英雄身份来自血缘、事迹、祭祀还是后世尊奉；
- 他与哪些神、圣物和神话事件相连；
- 死后是否神化，以及神化发生在哪一层材料中；
- 不同文本如何改变其结局。

英雄条目使用 hero.传统.名称 作为稳定 ID，并遵守 [三层正典](../00_foundation/canon.md)。

长廊也保存处在人、制度与神圣身份边界上的行动者。此类条目使用 actor.传统.名称，并明确说明为何不应误编为神座。

## 已开放档案

- [大禹](./yu-the-great.md)：以疏导洪水、驱逐蛇龙和恢复可定居土地建立人间秩序。
- [赫拉克勒斯](./heracles.md)：在神人血缘、艰苦功业、死亡影像与神化之间保持多重身份。
- [法老（王权角色）](./egyptian/pharaoh.md)：把具体历史国王与荷鲁斯身份、拉之子、玛阿特维护者和死后奥西里斯化分层保存，而不是制造一位抽象“法老神”。
- [吉尔伽美什](./mesopotamian/gilgamesh.md)：并列可能的历史王、苏美尔诗篇、古巴比伦版与标准巴比伦史诗。
- [恩奇都](./mesopotamian/enkidu.md)：保存荒野、人化、友谊和死亡，不把文明化写成无代价升级。
- [阿特拉哈西斯](./mesopotamian/atrahasis.md)：从人类创造、灾害控制到洪水后人口安排保存完整叙事链。
- [阿达帕](./mesopotamian/adapa.md)：以智慧、服从与错失不死呈现神人边界。
- [基尔塔](./ugaritic/kirta.md)：受埃尔梦示恢复王室，却因婚姻愿誓与疾病陷入继承危机。
- [达尼鲁](./ugaritic/danilu.md)：在城门为寡妇孤儿断案，又以父亲和亡灵英雄身份面对阿克哈特之死。
- [阿克哈特](./ugaritic/aqhat.md)：神匠弓的持有者，其拒绝、死亡与妹妹复仇线停在泥板断口。

大禹与赫拉克勒斯都以功业闻名，但功业的结构并不相同；法老条目则保存制度性神圣角色，不是第三位同类英雄。这些档案之间只能建立有证据边界的 motif-analogue 或王权比较。
