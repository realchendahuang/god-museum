# 法老（王权角色）

> “法老”在本馆不是一位神的名字，也不是所有埃及国王共享的固定人格；它是把历史统治者、荷鲁斯身份、拉之子、玛阿特维护者与死后奥西里斯化连接起来的王权角色。

## 编目判断

| 项目 | 内容 |
| --- | --- |
| 埃及语 | pr-ꜥꜣ 原意“大宫殿”，到新王国后期逐渐用于称呼国王本人 |
| 更早通称 | nswt-bjtj 等王衔；现代“法老”常被宽泛用于所有古埃及国王 |
| 类型 | royal_actor：处在人、制度与神圣身份之间的王权行动者 |
| 为什么不是神座 | 每位国王是具体历史人物；王权神性是称号、仪式和关系结构，不是一个跨朝代永生人格 |
| 为什么进入英雄长廊 | 王连接人间与诸神域，执行供奉、战争、建庙、纪念和丧葬转化 |

## 活着的王

### 多重王名

成熟的埃及王名系统包含荷鲁斯名、两女神名、金荷鲁斯名、上下埃及王名与拉之子名等部分。王名把具体统治者嵌入荷鲁斯、两地、太阳和王朝秩序。不同年代的格式与使用会变化，不能把五重王衔倒投到所有早期统治者。

### 荷鲁斯身份

王可作为活着的[荷鲁斯](../../02_deities/egyptian/horus.md)行动，使在位统治与奥西里斯—荷鲁斯继承结构相连。这是 historical-identification / ritual identity，不表示每位国王的个人记忆都被同一个荷鲁斯人格吸收。

### 维持玛阿特

王权图像把国王描绘为建庙、供奉、击败敌人和“献上[玛阿特](../../02_deities/egyptian/maat.md)”的人。实际每日仪式多由祭司代理执行，却以王的理想身份写入神庙。档案必须区分图像中的永恒王权动作与具体历史行政。

## 死亡后的王

金字塔文首先在古王国王室金字塔保存大量死后转化言说：王升入天空、加入星辰与太阳、与[奥西里斯](../../02_deities/egyptian/osiris.md)建立身份。后来的棺材文和亡灵书扩大并改变文本使用者与媒介，但不能简单说“原本只有法老能有来世，后来普通人才获得”。

王死后可以被称为奥西里斯，继任者承担荷鲁斯角色。这一结构帮助王朝连续，也无法阻止历史中的篡位、记忆删除、墓葬破坏和重新合法化。

## 国王不是一个人

未来每位国王都应单独建立历史档案，例如：

- 王名与在位年代采用哪套年表；
- 实际建造、战争、婚姻与行政证据；
- 与哪些神庙和祭司团发生关系；
- 死后祭祀、王表收录或除名情况；
- 后世文学、民族主义与流行文化如何重写其形象。

“法老下令”“法老相信”如果没有具体人名、时期和材料，默认属于不合格陈述。

## 人神边界

| 关系 | 类型 | 边界 |
| --- | --- | --- |
| 王—荷鲁斯 | historical-identification / office | 活王身份与王名结构；具体国王仍独立 |
| 王—拉 | kinship-as-kingship | “拉之子”等称号表达合法性，不是现代生物谱系 |
| 王—奥西里斯 | funerary-identification | 死后转化与继承结构，不删除墓主姓名 |
| 王—玛阿特 | performer / offerer | 王应维持并献上秩序，不拥有玛阿特 |
| 王—神像 | ritual service | 理想执行者；现实中由祭司按值日与职司代理 |

## 设定层

众神殿没有“法老神座”，只有一条王名长廊。每位国王以自己的名字、年代和证据进入；没有姓名的抽象法老只能作为制度影子出现。

当其他文明的神王、天子、圣王或祭司王进入比较时，本条目只提供“人如何承担神圣王权角色”的问题框架，不宣布制度同源。谁是神、谁是神之后裔、谁由天命授权，必须由各自传统回答。
