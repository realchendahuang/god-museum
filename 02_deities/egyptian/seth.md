# 塞特

> 塞特是不可预测、越界与破坏性力量的神格，也是某些时期的王权守护者和太阳船抗敌者；把他固定为“埃及邪神”会删掉大半历史。

## 身份

| 项目 | 内容 |
| --- | --- |
| 埃及学转写 | Stẖ；Seth / Set / Sutekh 反映不同语言与历史形式 |
| 核心关系 | 奥西里斯的兄弟与杀害者、荷鲁斯的叔父与王位竞争者 |
| 地方中心 | 努布特／Naqada 等；地方地位随政治与时代显著变化 |
| 图像 | “塞特兽”及其头部符号；现实物种对应没有定论 |
| 权柄张力 | 混乱与必要力量、王位威胁与边境保护、杀害者与抗击阿佩普者 |

## 原典层

### 争位者不是纯粹恶魔

在奥西里斯传统中，塞特杀害兄长并争夺王位；《荷鲁斯与塞特之争》又让他在诸神法庭、竞赛和身体冲突中持续挑战荷鲁斯。他可以欺骗、伤害和逾越边界，也可以提出自己拥有力量、资历和保护太阳船的能力。

文本并没有让所有神从开头就用同一个“邪恶”标签处理他。裁决迟延本身说明王权、力量和亲子继承之间存在真实张力。

### 被秩序使用的危险力量

太阳夜航和相关抗蛇图像中，塞特的强力可被用于攻击阿佩普。这里不是塞特突然变成温顺秩序神，而是危险力量被置于对付更彻底的宇宙阻断者的位置。

在边境、沙漠、风暴和外来力量的语境中，塞特既可代表威胁，也可成为王权需要驾驭的守护能量。拉美西斯时期王名 Sethi 及对 Sutekh 的尊崇表明其国家位置并非始终负面。

### 后期妖魔化

第一千纪的部分政治与祭祀变化使塞特更频繁地与被排斥的敌人、外来者和混沌联系。晚期资料不能倒投为早王朝以来三千年不变的“恶神本质”。

## 图像边界

所谓“塞特兽”具有长而弯的吻部、直立耳和分叉尾等形式特征，却没有获得学界一致同意的现代动物物种鉴定。God-Museum 把它记录为神圣图像语法，不把土豚、驴、豺等猜测写成确认事实。

## 整理层

塞特进入第八殿时提出一个难题：守护秩序是否可以借用本身危险的力量？进入第十一殿时则代表越界、诡计、异域和边境；进入第四殿的只是风暴面相。

他与北欧洛基、希腊提丰等角色最多建立有限 motif-analogue。反叛、怪物、风暴、诡计和末日敌对属于不同关系，不能因为现代作品都叫“反派”就建立同神关系。

## 设定层

塞特神座被允许保留矛盾：同一股力量可以杀害奥西里斯，也可以刺向阿佩普。众神殿不替他洗白，也不接受一张“善恶阵营表”取消真实历史中的祭祀与王权位置。

每当共殿需要以危险力量保护自身，塞特都必须说明力量的目标、期限与退出方式。争议不会阻止他发言，反而构成他入殿的理由。
