# 拉

> 拉既是太阳神名，也是创造、王权与每日更新的枢纽；拉—阿图姆、阿蒙—拉等组合必须作为有历史位置的复合关系保存。

## 身份

| 项目 | 内容 |
| --- | --- |
| 埃及学转写 | Rꜥ；Ra / Re 是现代语言中的常见写法 |
| 核心位置 | 太阳的神圣力量、创造者、昼夜航行者、王权之父 |
| 重要中心 | [赫利奥波利斯／Iunu](../../07_places/egyptian/heliopolis.md)及其他太阳祭祀网络 |
| 主要复合神格 | 拉—阿图姆、阿蒙—拉；复合不取消组成神的独立历史 |
| 主要危机 | 夜航受阻、[阿佩普](../../06_beasts/egyptian/apep.md)攻击、衰老与人类叛乱传统 |

## 档案导读

- **太阳不只是一颗天体：** 拉的光、热、运动、创造与王权关系在不同文本中组合方式不同。
- **日落不是永久死亡：** 夜航材料让太阳穿过杜阿特，与奥西里斯的死亡力量发生结合后再生。
- **复合名称不是别名表：** 拉—阿图姆与阿蒙—拉都是真实历史中的神学组合，应标作 syncretism。
- **不要写成唯一创世神：** 埃及创世表达还包括八位原初力量、普塔、赫努姆等不同地方与文本体系。

## 原典层

### 创造与显现

UCL 的材料综述把太阳神视为古埃及宗教中最主要的创造者之一。Rꜥ 是太阳之名；阿图姆由“完整、未分”一类意义解释为万物尚集中于创造者时的状态。创世不是一次机械制造，而是未分整体逐步展为舒、泰芙努特、盖布、努特以及后续神代。

苍鹭落上第一片土地、莲花从水面开放、第一丘从洪水显出，都是创造者出现的不同图像。它们能并存，不必选出一项作为“真实物理过程”。

### 白昼出行与夜间更新

《亡灵书》古代名称中的“白昼出行”与巴获得移动、加入太阳循环有关。《阿姆杜阿特》则按十二个夜间时段组织太阳船的行程：拉进入杜阿特、遭遇门、守卫和敌对力量，与奥西里斯之体结合，再以更新形态走向黎明。

[太阳夜航](../../08_myths/egyptian/solar-night-journey.md)因此不是简单的“拉每天打赢一条蛇”。它同时处理时间、死亡、尸体、再生与宇宙持续运行。

### 拉之眼与王权

《天牛之书》的毁灭人类段落把“拉之眼”呈现为可以从太阳神分出的女神行动力量。她以塞赫麦特的狂暴面相惩罚叛乱，又在染红啤酒计策中停止毁灭并回到哈索尔关联面相。这证明“眼”可以是关系、行动者与神格，不只是身体器官。

法老以“拉之子”等王权表达与太阳神相连，但每位历史国王不能因此被并入拉的神座。

## 整理层

拉同时打开第三殿、第一殿和第十二殿。分类不是复制三个太阳神座，而是从天体、创造和死后循环三个方向读取同一神座。

与其他文明太阳神建立 domain-analogue 前，必须分别证明：对方究竟是太阳本身、驾驶者、光的给予者、王权祖先还是季节神。仅有圆形日轮图像不足以建立身份关系。

## 设定层

拉的神座没有永恒正午。神座背面始终连着杜阿特，任何进入者都能看见光从衰老、黑暗、结合与战斗中重新产生。

阿蒙—拉或拉—阿图姆进入众神殿时，复合名称会在中央显现，两侧仍保留阿蒙、拉和阿图姆各自的档案门。众神殿承认复合神格的整体性，也拒绝用整体抹去形成它的历史。
