# 托特

> 托特让言语、文字、计算、时间与裁决成为可记录、可重复和可校正的秩序；他不是抽象的“全知之神”。

## 身份

| 项目 | 内容 |
| --- | --- |
| 埃及学转写 | Ḏḥwty；词形与朱鹭名称相关 |
| 核心中心 | Khemenu／赫尔莫波利斯（“八之城”） |
| 常见图像 | 朱鹭首人身、朱鹭或狒狒；图像需依铭文辨名 |
| 核心职能 | 书记、计算、月与历法、神圣言语、裁决记录、医治荷鲁斯之眼 |
| 希腊认同 | 与赫耳墨斯发生 historical-identification，后续形成 Hermes Trismegistus 接受史 |

## 原典层

### 文字与计算

UCL 地方材料把托特称为知识和书写之神，并记录其主要神庙位于 Khemenu 与 Baqliya。Khemenu 意为“八”，使托特与八位原初力量的地方解释相连；但研究者不能反过来断言所有“八神创世”思想都只在这座城市一次发明。

托特掌握文字不等于“替人类发明所有文明”。他更常以诸神书记、语言和数字的操作者、时间和月份的计算者出现。每项发明主张需要具体文本。

### 修复与调停

《亡灵书》17 的神学注释把荷鲁斯之眼的受损与修复联系到托特。修眼使托特的知识具有身体和政治后果：恢复完整也帮助继承秩序继续。

在荷鲁斯与塞特争议及其他神界法庭材料中，托特可以书写、传达、调停或提出方案，却不总是最终决定者。知识进入政治时仍受诸神权力结构制约。

### 审判记录

《亡灵书》125 图像常让托特站在天平旁记录或宣告称心结果。记录不是被动抄写：文字使一次衡量成为可被法庭承认的裁决。但衡量尺度来自玛阿特，天平由阿努比斯处理，主持权属于奥西里斯审判体系。

## 月与时间

托特的月亮关系涉及计时、月相和历法调节。创世整理中，他帮助取得年外五日的故事属于后期叙事链，不能把它写成现代天文学史。进入第三殿和第十殿的是时间组织权柄，不是说托特等于月球本身。

## 设定层

托特掌管众神殿关系类型的书写，却不能单方面决定神座身份。他可以指出一条关系应为 syncretism 而非 same-identity，可以记录争议，却必须让材料本身留下异文。

他的书库没有“世界真相总卷”。每份记录都保留抄写者、媒介、缺损和时代；知识的可靠性来自可追踪，不来自神名自动保证无误。
