# 阿努比斯

> 阿努比斯守护尸体、防腐过程、墓地边界与审判通路；“冥界之王”在奥西里斯传统成熟后不是对他的准确概括。

## 身份

| 项目 | 内容 |
| --- | --- |
| 埃及学转写 | Jnpw；希腊语形式 Anoubis 进入现代语言 |
| 核心职能 | 防腐、裹尸、尸体与墓室保护、引导死者、审判天平 |
| 早期地位 | 在古王国丧葬材料中已十分重要，部分权柄早于奥西里斯广泛扩展 |
| 图像 | 卧伏黑色犬科动物或犬首人身；“胡狼”是通行描述，不等于现代动物学已完全确定 |
| 关键文本与器物 | 金字塔文、亡灵书 125 与 151、印章、棺椁和魔砖 |

## 原典层

### “在防腐之处者”

UCL 地方材料保存一枚写有“阿努比斯，在防腐之处者”称号的印章，并把他概括为年轻王权力量与防腐之神。防腐不是单一技术动作，而是让身体重新获得完整、身份和神圣可接近性的仪式过程。

与尸体有关的不同称号会强调“在神棚中者”“在其山上者”“西方人之首”等位置。称号与地方神的关系必须逐项辨别，不能都自动视为同一个常驻职位。

### 审判中的职司

《亡灵书》125 的常见图像让阿努比斯引导死者并操作或检查称心天平。玛阿特之羽提供秩序尺度，托特记录结果，奥西里斯主持死者法庭，阿米特等待不利裁决。阿努比斯不是凭个人喜好决定善恶的独裁判官。

### 墓室保护

《亡灵书》151 与四方魔砖实践把阿努比斯形象、杰德柱、火炬和人形护符置于墓室不同方位。芝加哥大学馆藏显示，真实墓葬中的砖、护符、文字和方位并不总严格照纸草指令执行。实践差异本身要进入档案。

## 谱系边界

阿努比斯的父母在不同时期和文本中有多种说法：可与拉、奥西里斯、奈芙蒂斯等建立不同亲缘。God-Museum 暂不选一条作为唯一出生版本；每次 kinship 关系必须带上文本节点。

他与 Wepwawet 都可能呈犬科形态且关联道路、墓地或王权，却不能仅凭外貌写成 same-identity。地方称号、图像和祭祀证据应先分开。

## 设定层

阿努比斯负责第十二殿所有“尚未完成转化”的门槛：身体仍需处理、名字仍需确认、道路仍需开启、衡量仍未结束。他没有权把其他文明死者自动木乃伊化，也不会把所有死后审判改成称心。

在众神殿里，他最重视的不是死亡本身，而是转化是否按其传统所承认的程序完成。任何被粗暴搬运、匿名陈列或失去墓葬语境的遗体都会使他的神座发出警告。
