## 新建档案

请从 GitHub 复制 [_template.md](https://github.com/realchendahuang/god-museum/blob/main/02_deities/_template.md)，确定稳定 ID 后再写正文。

一份正式神明档案至少要回答：

1. 这位神属于哪个具体传统与时期？
2. 哪些名字是异名，哪些只是翻译？
3. 他的权柄由哪些材料直接支持？
4. 不同文本中的面相如何变化？
5. 与其他神的关系属于哪一种关系类型？
6. 哪些内容来自 God-Museum 原创？
