# 赫拉

> 赫拉是王后、婚姻与合法结合的神座；她在史诗中的冲突不能把其祭祀身份缩减成“善妒的妻子”。

## 身份

| 项目 | 内容 |
| --- | --- |
| 原文名 | Ἥρα |
| 主要身份 | 克洛诺斯与瑞亚之女、宙斯配偶、诸神之后 |
| 核心关系 | 婚姻、王后权威、女性生命阶段与城邦祭祀 |
| 主要锚点 | 《神谱》921–923；《荷马体赞歌》12 |

## 原典层

《神谱》921–923 将赫拉置于宙斯婚配和后代谱系中，并列出赫柏、阿瑞斯与厄勒梯亚。《致赫拉》称她为克洛诺斯之女、诸神之后，并强调其与宙斯共同受到尊崇的高位。

这些短文本能确认王后和婚姻位置，却不足以概括各城邦的 Heraia、称号与生命礼仪。史诗里她与宙斯的冲突属于神圣权力关系，不应成为全部身份。

### 争议与空白

- 迈锡尼材料、阿尔戈斯和萨摩斯祭祀需按地点分别建源；
- “婚姻女神”不表示她只保护私人家庭；婚姻同时涉及王权、继承与城邦秩序；
- 罗马朱诺与赫拉属于历史认同和重组，不默认完全同一。

赫拉进入主权与婚姻相关展区，不以宙斯附属神身份入殿。她与其他婚姻神的比较必须保留王后和城邦政治层。

## 设定层

赫拉的神座承认关系是否被公开、见证和维持。她不能强迫产生爱情，却能揭露一项联盟是否试图借神名获得合法性。

### 可展开的故事

- 当跨文明神婚被当作政治条约时，赫拉要求双方先说明各自传统中的婚姻究竟意味着什么。
- 一位统治者企图只借宙斯雷霆取得王权，赫拉以王后席位否决未被共同秩序承认的统治。
