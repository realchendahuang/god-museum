# 珀耳塞福涅

> 珀耳塞福涅同时是被夺走的女儿、返回母亲身边的科瑞和冥界的王后；任何单一身份都不足以覆盖她的往返。

## 身份

| 项目 | 内容 |
| --- | --- |
| 原文名 | Περσεφόνη；赞歌中也称 Kore“少女” |
| 母亲 | [得墨忒耳](./demeter.md) |
| 核心结构 | 被带入冥界、食下石榴籽、周期性返回、冥界王后 |
| 主要锚点 | 《荷马体赞歌·致得墨忒耳》 |

## 原典层

赞歌开头让珀耳塞福涅采花时被黑帝斯带走，宙斯此前已同意婚配。她的呼喊被母亲和部分神祇听见。重逢时，她叙述自己如何食下石榴籽，由此形成一年中在冥界与母亲之间分配时间的安排。

文本既保存被迫分离，也保存她后来作为冥界统治者的位置。她不是一粒被动种子的拟人化，也不能只被称为黑帝斯妻子。

### 争议与空白

- 石榴籽数量在后世版本中变化；当前条目以赞歌为准；
- Kore、Persephone 及地方称号需要祭祀史分层；
- 现代“自愿选择冥界”改写不能冒充赞歌叙事。

## 设定层

珀耳塞福涅拥有两把不能同时使用的钥匙：一把开启归返，一把确认冥界王后的裁决。她的限制不是力量不足，而是每次越界都必须面对两套关系的真实要求。
