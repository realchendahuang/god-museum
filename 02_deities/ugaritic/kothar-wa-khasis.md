# 科塔尔—哈西斯

> 科塔尔—哈西斯以“熟练而智慧”的双重名字制造能改变神战的武器、建造确认王权的宫殿，也体现乌加里特港口如何把外来技艺写进神话。

## 身份

| 项目 | 内容 |
| --- | --- |
| 名称 | Kôṯaru-wa-Ḫasīsu，常译“巧匠而智慧者”；双名是称号式神名，不拆成两位神 |
| 居所／来源 | Kaptaru 与 Ḥkpt 等地点常被联系克里特／卡夫托与埃及 Memphis，具体地理辨认需谨慎 |
| 核心作品 | 《巴力循环》《阿克哈特史诗》 |
| 主要作品物 | [巴力双棒](../../05_relics/ugaritic/baal-clubs.md)、巴力宫殿、[阿克哈特之弓](../../05_relics/ugaritic/aqhat-bow.md) |
| 核心位置 | 神匠、建筑师、命名武器者、技术方案提出者 |

## 原典层

巴力与雅姆战斗时，科塔尔不仅打造武器，还为两件棒命名并指示它们如何行动。名字像命令一样使武器识别目标，战斗胜利因此依赖工艺、语言和使用者配合。

巴力建宫时，科塔尔提出开窗，巴力起初拒绝，后来同意。建筑不是中性容器：窗决定雷声、云雨和王权如何从宫殿进入世界。

《阿克哈特史诗》中，科塔尔来到达尼鲁之家，把弓赠给新生／年轻的阿克哈特。弓随后引发阿纳特欲求、拒绝和谋杀。制造者没有在现存文本中预定悲剧，圣物进入社会关系后产生超出工艺本身的后果。

## 外来地点与传播

科塔尔往返的地点让学界联系爱琴海与埃及工艺中心。乌加里特本身是多语言港口，这种地理想象有历史环境基础；但把 Kaptaru 直接等于现代地图上的单一岛屿，或宣布科塔尔就是赫淮斯托斯／普塔，均需额外证据。

## 整理层

科塔尔主要进入第九殿，并连接战争和道路。他证明文明通道不只运输神名，也运输金属、建筑知识和工匠声望。

## 设定层

科塔尔的工坊有通向海上的门，但每件材料都标明来源。未经历史道路证明的“同一神匠”，无法共享工具台。

他制造武器却不替持有者承担全部伦理；宝库会同时记录制造意图、实际用途和失控后果。
