# 沙普舒

> 沙普舒不是只在天空发光的背景太阳；她能穿行神域与死亡边界、协助阿纳特寻找巴力，并在最终冲突中以埃尔权威迫使莫特让步。

## 身份

| 项目 | 内容 |
| --- | --- |
| 名称 | Šapšu，乌加里特语太阳神名；在乌加里特为女神 |
| 常见称号 | “诸神的灯／火炬”等，具体翻译随文本 |
| 核心位置 | 太阳路线、照见远方、传讯、陪伴死者／亡灵仪式、调停 |
| 主要关系 | 阿纳特的行动伙伴、巴力与莫特冲突的中介 |
| 性别边界 | 同源太阳词在其他闪米特传统中神格性别可不同，不能由语言直接统一 |

## 原典层

巴力被宣布死亡后，阿纳特向沙普舒求助。太阳每日跨越天空与地界的能力，使她成为寻找、移动和连接领域的关键者。她协助带回／处理巴力身体的具体动作因残缺行与译法存在差异。

在《巴力循环》结尾，巴力与莫特重新对战而难分胜负。沙普舒告诉莫特：若继续对抗，埃尔将转而支持巴力／推翻其位置。莫特因此屈服。最终解决不是太阳以武力击杀死亡，而是传讯与权威判断改变政治选择。

祭祀与咒语文本也把沙普舒连接到蛇咬、神群和亡灵边界。叙事职能不能完全覆盖其仪式角色，未来需逐 KTU 条目扩展。

## 整理层

沙普舒进入第三、十一、十二殿。与沙马什、拉、苏利耶等太阳神比较时，不能默认太阳神为男性，也不能把全见、司法、复生和引路能力打包成通用太阳职能。

## 设定层

沙普舒的神座每天穿过众神殿一次，但她只照亮有文本道路相连的房间。光能发现门，不等于拥有门后的王权。

她传达埃尔的判断时，话语保留发言者和听者，不被改写成太阳女神自己的最终法令。
