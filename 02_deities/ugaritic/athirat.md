# 阿舍拉／阿西拉图

> 阿西拉图既是生育神群的母性高位，也是能接受礼物、判断政治时机并说服埃尔的神议行动者；“埃尔妻子”远不足以概括她。

## 身份

| 项目 | 内容 |
| --- | --- |
| 乌加里特名 | ʾAṯiratu；现代常用 Athirat |
| 常见称号 | rbt ʾaṯrt ym，常译“海之女主阿西拉图”，ym 的关系与译法仍有讨论 |
| 母性称号 | qnyt ʾilm，常译“诸神的生育者／母亲” |
| 配偶 | [埃尔](./el.md) |
| 核心位置 | 神群母性、宫殿斡旋、王位候选安排、祭祀名单中的高位女神 |

## 原典层

《巴力循环》建宫段落中，巴力没有直接取得埃尔同意。科塔尔准备礼物，阿纳特与阿舍拉发生接触，阿舍拉最终前往埃尔居所，为巴力拥有宫殿进行斡旋。她能读懂礼物和权力时机，不是被动传话者。

巴力被宣布死亡后，埃尔请阿舍拉从其众子中选择能继任者。阿斯塔特试坐王位而不合适。阿舍拉因此参与神权空缺的制度处理。

她与“七十子”或神群数量的关系常见于研究重构，具体数字、名单和父母结构必须按不同文本记录，不能用一个家庭合照替代神名表。

## 海、树与后续 Asherah

称号中的 ym 常被理解为“海”，但阿舍拉并非因此等同[雅姆](./yam.md)或普通海洋女神。后世希伯来铭文与圣经文本中的 ʾašerah 可以指神名、祭祀物／木质象征或遭批判的崇拜对象，语义和年代必须分别研究。

乌加里特 Athirat 与后续 Asherah 可建立 transmission / linguistic-continuity，不把所有“圣树”“生命树”图像自动识别为她。

## 关系

| 对象 | 类型 | 边界 |
| --- | --- | --- |
| [埃尔](./el.md) | marriage / council-partnership | 配偶与神群父母位置，不抹去女神政治行为 |
| [巴力—哈杜](./baal-haddu.md) | mediation | 为其建宫向埃尔斡旋；不是巴力母亲的无争议固定关系 |
| [阿纳特](./anat.md) | cooperation / negotiation | 同为建宫行动者，策略与职能不同 |
| “阿舍拉诸子” | kinship-group | 名单、数量与个体身份需按文本，不制造统一七十神表 |

## 整理层

阿舍拉连接第七殿和第二殿，说明生育母性可以直接进入神权制度。与赫拉、伊西斯、女娲或其他母神只能比较配偶、母职、议庭和地方祭祀的不同组合。

## 设定层

阿舍拉的神座不是一棵被误认的树，而是一条从礼物桌通往议庭的路。任何请求若只把她当作埃尔的关系入口，都会在路中途失去声音。

她允许后世名字与自己建立通道，却要求每块铭文说明：写的是女神、象征物，还是反对者眼中的祭祀实践。
