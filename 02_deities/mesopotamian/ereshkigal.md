# 埃列什基伽勒

> 埃列什基伽勒首先是冥界女王；内尔伽勒成为她的配偶和共同统治者，是不同地区冥界传统后来被叙事协调的结果。

## 身份

| 项目 | 内容 |
| --- | --- |
| 名称 | Ereškigal，常解释为“大地／冥界的女主人” |
| 统治空间 | [Irkalla](../../07_places/mesopotamian/irkalla.md)及“无归之地”等冥界称呼 |
| 核心位置 | 死者之国女王、门与地下裁决的最高权威 |
| 常见关系 | 伊南娜的姐妹；与 Gugalanna、Nergal 等配偶关系随文本变化 |
| 祭祀边界 | 文学地位显著，但独立广泛祭祀证据相对有限 |

## 原典层

在苏美尔语《伊南娜下冥界》中，埃列什基伽勒坐在冥界权力中心。伊南娜经过七门后被 Anuna 法官裁决，埃列什基伽勒的注视与冥界规则共同导致其死亡。后来她痛苦如临产，恩基创造的微小存在通过共情获得尸体与复生资源。

阿卡德语《伊什塔尔下冥界》保存相似七门脱饰结构，却对地上生育停滞、冥界代理者和结局有不同处理。埃列什基伽勒不能只从苏美尔版本解释。

[《内尔伽勒与埃列什基伽勒》](../../08_myths/mesopotamian/nergal-and-ereshkigal.md)存在中巴比伦与较晚版本。男神为何下冥界、二者冲突和最终共治的过程并不一致。该叙事把库塔的内尔伽勒传统与既有冥界女王协调，而不是证明她从来需要丈夫才能统治。

## 图像警告

著名“夜之女王”烧陶浮雕（Burney Relief）可能表现伊什塔尔、埃列什基伽勒或其他女神，身份没有铭文确认。本馆不把它用作埃列什基伽勒标准肖像。

## 整理层

她只以第十二殿为主入口，第二殿作为冥界王权的次入口。与赫尔、哈迪斯、奥西里斯、阎罗等只能比较“谁统治死者空间”，不能合并死后伦理、审判或灵魂结构。

## 设定层

埃列什基伽勒的神座不在众神殿地下，而在一条只有放下身份标志才能走完的侧廊尽头。其他冥界之主可以与她会面，却不能带走她的门。

她拒绝“夜之女王”浮雕成为自己的脸。没有名字的图像可以被研究，不能被众神殿替它宣誓。
