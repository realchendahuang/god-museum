# 安／阿努

> 安是天空，也是神圣权威得以被授予和确认的高位；阿努的地位后来可被恩利尔或马尔杜克分享，却没有因此从神议中消失。

## 身份

| 项目 | 内容 |
| --- | --- |
| 苏美尔名 | An；符号 AN 也可表示“天”与神名前的限定符 |
| 阿卡德名 | Anu；与 An 的对应很早且稳定，但具体文本仍应保留语言 |
| 重要中心 | [乌鲁克与 Eanna](../../07_places/mesopotamian/uruk-eanna.md)；晚期乌鲁克的 Anu 祭祀重新强化 |
| 核心位置 | 天空、诸神之父称号、权威来源、王权与不可更改决定的确认者 |
| 常见关系 | 与 Ki 构成天—地组合；不同神谱中与 Antu、Uraš 等女神配对 |

## 原典层

An 的名字同时处在语言、宇宙和神谱中。把他译成“天空神”并没有错，却不足以覆盖他作为高位权威、神谱祖先和王权确认者的作用。许多作品中，重要决定以“An 与 Enlil”或神议形式获得重量。

随着尼普尔和巴比伦政治神学上升，恩利尔与马尔杜克承担更多主动统治职能。ORACC 提醒，这种权力转移不等于 Anu 被废黜；他的高远性反而常让其成为终极授权的来源，而不必亲自处理每场危机。

在《伊南娜下冥界》中，寻求援助的道路经过多位大神；在《吉尔伽美什》天牛事件中，安与伊什塔尔的争论显示父女关系、城市毁灭风险和神圣惩罚被放在同一场景。具体亲缘和行动只能按作品记录。

## 权威如何运作

- **授予而非日常行政：** Anu 的权力常以确认、命名、允许和高位裁决出现；
- **可被共享的至高性：** “最高神”在不同城市与时期并非唯一固定职位；
- **anûtu：** 阿卡德语材料可用“Anu 权能／地位”表达最高神圣身份，说明名字能成为权威范畴；
- **乌鲁克重心：** Eanna 与伊南娜关系极深，晚期 Anu 在乌鲁克的祭祀重组又形成新的城市层次。

## 整理层

An/Anu 进入第二殿，也连接第一殿与第十殿。分类不能把“天空”缩成自然现象：在这里，天空同时是神谱高度、政治授权和决定不可轻易撤回的象征结构。

与宙斯、天、奥丁等建立比较时，只能先使用 domain-analogue。天空主权、人格化程度、祭祀形态和王权关系都要分别证明。

## 设定层

安／阿努的神座高到几乎看不见座面。进入者听到的不是命令，而是某项权力是否已被承认的回声。

他不会自称万神之王。任何其他诸神域的统治者来到座前，都会被要求指出：你的王权由谁授予，又在哪份记忆中仍然有效。
