# 伊南娜／伊什塔尔

> 伊南娜／伊什塔尔同时进入爱欲、战争、金星与王权，她的矛盾不是后人拼坏的人设，而是跨语言、城市与帝国保存下来的神格强度。

## 身份

| 项目 | 内容 |
| --- | --- |
| 苏美尔名 | Inana / Inanna |
| 阿卡德名 | Ištar；原有闪米特女神背景，后来与 Inana 深度认同 |
| 重要中心 | [乌鲁克与 Eanna](../../07_places/mesopotamian/uruk-eanna.md)；另有多个地方祭祀中心与面相 |
| 天体 | 金星的晨星与昏星显现；天体关联不等于现代天文学人格化说明 |
| 常见象征 | 八角星、狮、武器簇；图像身份需结合铭文与器物 |

## 原典层

ORACC 将爱欲、战争、政治权力与金星列为这一神格最持久的轴线。爱与战并非两个互相取消的版本：女神可以激发欲望、颠倒身份、使王获得合法性，也能站在兵器上、摧毁城市并使军队溃败。

[《伊南娜与恩基》](../../05_relics/mesopotamian/me.md)让她把 me 从埃里都带往乌鲁克，表现城市文明力量的获得。[下冥界](../../08_myths/mesopotamian/descent-to-the-netherworld.md)则让她在七门前逐件失去权力标志，遭裁决、死亡和替代规则约束。

《吉尔伽美什》标准巴比伦版中，伊什塔尔向吉尔伽美什求婚、遭拒后请求安释放天牛。英雄的拒绝清点女神旧爱结局，具有史诗人物的修辞立场，不能直接变成研究者对整套女神祭祀的客观评价。

## 神圣婚姻的争议

赞歌和王权文本可把女神、杜穆济与国王连接在“神圣婚姻”语言中。现代研究对是否每年实际举行包含性交的固定仪式保持争论。本馆区分诗歌、王权表演、祭仪推测和流行叙事，不把争议假说写成已确认现场。

## 性别与边界

献给伊南娜／伊什塔尔的文本涉及性别角色改变、非典型祭司身份与社会秩序翻转。相关术语需逐词、逐时期研究，不用现代身份类别一对一覆盖，也不把这些材料删成单纯“爱情女神”背景。

## 整理层

Inana 与 Ištar 标为 historical-identification / syncretism。讲苏美尔下降文本时用伊南娜，讲阿卡德语下降文本时用伊什塔尔；跨时段神格档案才合写。

她连接第七、八、三、二殿，正好证明十二殿不是性格分类。爱欲、战争和王权在古代城市政治中可以由同一神圣力量承担。

## 设定层

伊南娜／伊什塔尔的神座既有晨星的冷光，也有兵器反射的光。她拒绝让众神殿替她选择“爱”或“战”的一种面孔。

通向冥界的七件权力标志保存在座后，却不再被当作万能装备。每次她走向一扇不属于自己的门，门仍有权要求她放下一件。
