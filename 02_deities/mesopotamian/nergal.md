# 内尔伽勒

> 内尔伽勒掌管的不是安宁死亡，而是战争、瘟疫、疾病与灼热造成的暴死；他进入冥界共同王权有清楚的历史形成过程。

## 身份

| 项目 | 内容 |
| --- | --- |
| 重要中心 | 库塔（Kutha）的 E-meslam |
| 核心位置 | 战死、疫病、毁灭性热力、冥界力量 |
| 神格关系 | 与 Meslamtaea、Erra 等发生不同程度认同或融合 |
| 常见象征 | 狮、武器、复合神徽；不能凭武装神像直接识别 |
| 配偶 | [埃列什基伽勒](./ereshkigal.md)，但共治是后起传统 |

## 原典层

ORACC 将 Nergal 与战争造成的死亡、瘟疫和疾病相连。正午太阳的毁灭性热也进入其职能，这与乌图／沙马什作为光、司法和每日路线的太阳神职不同；同一自然现象可以在不同神格中承担不同面向。

Nergal、Erra 与 Meslamtaea 不能无条件列为三个完全同义名字。Erra 在《埃拉史诗》中表现毁灭、战争和瘟疫的独特叙事人格；Meslamtaea 与库塔地方传统、双神关系和早期冥界神学相连。后世融合真实存在，前史也必须保存。

[《内尔伽勒与埃列什基伽勒》](../../08_myths/mesopotamian/nergal-and-ereshkigal.md)让他从天上神议进入冥界，与女王冲突并最终取得共治位置。中巴比伦版与较晚版本在使者、礼仪冒犯、武力和婚姻安排上不同。

## 《埃拉史诗》的边界

《埃拉史诗》把毁灭冲动、城市灾难、马尔杜克神像与秩序暂时失守编织在一起。不能把文学中的全球性破坏简单当作某次历史战争实录，也不能因 Erra 与 Nergal 融合就把每一行自动归给同一人格状态。

## 整理层

内尔伽勒连接战争、死亡与疾病，提醒本馆：医疗与生命之殿不能只收治愈神，也要记录疾病如何被神圣化。与阿瑞斯、阿努比斯、塞特等比较必须拆开战争方式、死亡类型与冥界权力。

## 设定层

内尔伽勒的神座表面有兵器划痕和疫火烧痕，却没有为“死亡之神”提供通用王冠。他拥有的每一种死亡都标明原因。

进入埃列什基伽勒的门后，他可以共治；回到众神殿，他仍必须说明这项权力来自哪一个版本。
