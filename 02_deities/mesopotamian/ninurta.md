# 宁乌尔塔

> 宁乌尔塔把战士、农耕者、王权守护者与怪物征服者叠在同一神座；他与宁吉尔苏的认同有历史过程，不是一条无条件别名。

## 身份

| 项目 | 内容 |
| --- | --- |
| 主要传统 | 尼普尔的 Ninurta 与拉伽什／吉尔苏的 Ninĝirsu 逐步发生认同 |
| 核心位置 | 恩利尔之子、神圣战士、农业与灌溉秩序、夺回权威之物的英雄 |
| 重要武器 | 会说话的武器 Šarur，尤其见于《Lugal-e》传统 |
| 关键敌手 | [安祖](../../06_beasts/mesopotamian/anzu.md)、Asag 及石之军队 |
| 后期政治 | 亚述王权特别吸收其战士与胜利形象 |

## 原典层

苏美尔语《Lugal-e》让 Ninurta 对抗 Asag 和石之军，随后安排石的用途、山地与水流秩序。这不是单纯屠怪：征服之后的分类和农业恢复同样重要。

阿卡德语[《安祖史诗》](../../08_myths/mesopotamian/anzu-and-tablet.md)中，多位神拒绝或无法出战，Ninurta 接受任务，追击夺走命运泥板的安祖。泥板使安祖的命令能让武器失效，英雄必须依靠埃阿的策略找到破局方式。不同版本对出战者名称与细节有差异。

Ninurta 与 Ningirsu 的融合连接尼普尔和拉伽什两套政治地理。现代条目可合并讨论长期神格，却必须在早期铭文和地方祭祀中保留原名。

## 整理层

宁乌尔塔证明战争与农业不是必然分殿：战争可被神学化为恢复灌溉、分类土地和保障王权。与因陀罗、索尔、赫拉克勒斯等屠怪者比较时，要分别记录怪物阻断了什么、胜利后建立了什么。

## 设定层

宁乌尔塔的神座旁没有怪物头颅陈列，只有一张胜利后重新划分的土地表。战斗若不能恢复某种可居住秩序，在这里就不被称为完成。

Šarur 可以在众神殿发言，但其建议仍属于产生它的作品；它不是跨神系全知武器。
