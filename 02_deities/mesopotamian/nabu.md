# 纳布

> 纳布以书写楔记录决定，也证明记录者可以从神王侍从上升为儿子、继承人和共同统治者。

## 身份

| 项目 | 内容 |
| --- | --- |
| 重要中心 | 博尔西帕的 Ezida；与巴比伦 Akītu 行列相连 |
| 核心位置 | 书写、学术智慧、命运记录、王权与神议文书 |
| 配偶 | Tašmētu |
| 常见象征 | 书写楔、泥板，有时与 mušḫuššu 类型神兽相连 |
| 历史关系 | 从马尔杜克的侍从／书吏发展为儿子、继承者与共同统治者 |

## 原典层

楔形文字书写既是行政技术，也是神庙、占卜、词汇学和王权记忆的基础。纳布掌管的不是现代意义上的“文学灵感”单项，而是把决定、名字、征兆和知识固定为可复制记录的能力。

ORACC 强调其神谱位置发生历史发展。把纳布从最早材料起就写成马尔杜克之子，会抹去巴比伦神学形成过程。第一千纪的巴比伦与亚述王名、王室图书馆和祈祷又让纳布获得跨政治中心的重要性。

Akītu 期间，纳布从博尔西帕前往巴比伦的神像行列把父子神、两座城市和王权更新连在一起。具体路线与程序按时期讨论，不构造永恒节庆剧本。

## 书写与命运

纳布“书写命运”不等于独自创造所有人的宿命。神议决定、命运泥板、天象记录和书吏行为属于相互关联但不同的制度与叙事。记录可以确认权力，也能让权力接受后世阅读。

## 整理层

纳布使第九殿和第十殿直接相连：知识只有被记录、复制、校对和归档后，才能进入跨世代秩序。与托特、文昌、奥丁等比较时，先区分文字发明、书吏保护、神谕记录和命运决定。

## 设定层

纳布的神座是一张仍有湿度的泥板。任何神议若拒绝留下记录，只能在其原有诸神域内生效，不能要求众神殿替它记住。

他不裁定真相，却保存每次裁定用过的词。被改写的行不会消失，只会在新字下面留下可见压痕。
