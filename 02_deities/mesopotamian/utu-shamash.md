# 乌图／沙马什

> 太阳每日看见道路、契约与隐藏之事，因此乌图／沙马什既带来光，也成为正义、真理、占卜和远行者的见证者。

## 身份

| 项目 | 内容 |
| --- | --- |
| 苏美尔名 | Utu |
| 阿卡德名 | Šamaš，源自闪米特语太阳词根；与 Utu 形成稳定认同 |
| 重要中心 | 西帕尔与拉尔萨的 E-babbar，“白屋／光明之屋” |
| 核心位置 | 太阳运行、司法、发现隐情、占卜、穿越边界与保护旅人 |
| 常见象征 | 太阳圆盘、从肩升起的光线、锯齿状器具、杆与环 |

## 原典层

太阳从东方显现、横越人间并在西方进入地下的宇宙路线，使乌图／沙马什能同时看见生者活动、远方道路和死者世界。苏美尔文学中的乌图帮助英雄穿越山地，阿卡德传统中的沙马什则是祈祷、法律与占卜的重要神圣见证。

“正义之神”不是一套现代法典人格化。神能使隐藏的案件显明、保护受欺者、见证誓言，也通过占卜把神意带给王与祭司。具体法律条文的颁布者仍是历史国王，不能把《汉谟拉比法典》全文称为沙马什亲自制定。

BM 91000“沙马什泥板”出自西帕尔：神坐于神龛，太阳圆盘与祭祀人员共同出现。[杆与环](../../05_relics/mesopotamian/rod-and-ring.md)是图像核心，却没有唯一无争议解释。

## 史诗中的援助

在胡姆巴巴远征传统中，太阳神与吉尔伽美什的请求、道路和自然力量相连。标准巴比伦版的具体神助安排与苏美尔诗篇并不完全相同，不能合成一个“太阳神施放十三道风”的固定技能说明。

## 整理层

本神座同时连接太阳、正义、占卜和道路。与其他太阳神比较时，必须问：太阳为何获得司法能力？是全见、王权、生命、时间还是冥界路线在起作用？

《汉谟拉比法典》石碑图像常被通俗称为“沙马什把法律交给汉谟拉比”，但图像身份、姿势和法典作者关系应在器物专页中谨慎展开，本页不把该解读当作唯一事实。

## 设定层

乌图／沙马什的神座每天从殿的一端走到另一端，任何案件只有在光线完整穿过其证据后才会被听见。

他不把自己的正义扩展成万神统一法。他只要求所有跨文明裁决先说明：谁看见了，谁作证，谁有权解释沉默。
