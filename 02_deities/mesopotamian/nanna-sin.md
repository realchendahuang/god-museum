# 南纳／苏恩／欣

> 月的盈亏使南纳／欣成为历法、月份、占卜与誓言的尺度；他的神座也连接乌尔王权、祭司职位和跨城传播。

## 身份

| 项目 | 内容 |
| --- | --- |
| 名称 | 苏美尔 Nanna；Suen / Sîn 的语言与书写关系复杂，按文本保留形式 |
| 重要中心 | 乌尔的 E-kiš-nu-gal；后期哈兰也是重要月神中心 |
| 核心位置 | 月亮、月份、时间测量、占卜、誓言与夜间秩序 |
| 常见象征 | 月牙、牛角联想、牛与舟；图像需按器物确认 |
| 著名祭司 | 阿卡德王萨尔贡之女恩赫杜安娜任乌尔月神高级女祭司 |

## 原典层

月相直接参与月份和祭仪时间的组织。南纳／欣不是抽象“夜神”：可见的盈亏、月牙如角的形态和夜间行程使其与牧群、丰饶、占卜和秩序相连。

乌尔王朝的建筑、赞歌和高级女祭司职位使月神成为城市政治的核心。恩赫杜安娜的名字与一批苏美尔文学作品相连，但她是否为所有传世文本的单一历史作者、作品经历何种抄写重编，现代研究仍有争论。本馆不把传统归属直接变成无争议签名。

在第一千纪，巴比伦与亚述政治对月神中心的利用、哈兰祭祀及国王母亲阿达德-古皮相关铭文，展示神座能跨城市获得新的政治生命。不能把乌尔与哈兰写成同一时期的单一总部。

## 整理层

Nanna、Suen、Sîn 不作为三个重复神座；正文却保留每种语言和时期的命名。与嫦娥、塞勒涅、托特等月亮相关形象只能先建 domain-analogue，神格性别、历法制度与叙事位置都不同。

牛角与月牙的视觉相似可以支持图像联想，不能证明所有牛形神像都是南纳／欣。

## 设定层

南纳／欣的神座按月相改变可见面积，却从不真正缺失。每一道誓言在这里都会获得日期；没有日期的永恒承诺被视为尚未进入人间。

乌尔与哈兰各自开一扇门，两门不会叠成一座城市。月光可以同时照见它们，历史不能。
