# 十二座权柄大殿

权柄大殿是跨文明阅读诸神的入口，不是新的神系，也不是力量排行榜。

每位神只有一个神座，却可以通过多个权柄进入多座大殿。大殿负责提出共同问题，神明档案负责保存差异。

## 入殿规则

- 进入大殿表示拥有相关权柄，不表示全部身份都由该权柄定义。
- 同一位神可以进入多座大殿，但只维护一个神明档案。
- 大殿中的并列默认是 domain-analogue，不自动建立同源或融合关系。
- 无法归类的神不被硬塞进现有结构；分类应当为材料让路。
