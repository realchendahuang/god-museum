# 静默长廊

静默长廊保存无法按照普通神明档案完成的存在：

- 只剩名字的神；
- 只在一块铭文或残片中出现的神；
- 身份被后世神系吸收、但原貌已无法恢复的神；
- 所属语言、文明或祭祀已经失传的神；
- 只能确认曾被祭祀、却不能确认职能的存在。

## 原则

未知不是可以随意补写的空白。

静默条目使用 fragmentary 状态，明确列出：

- 现存证据；
- 可以确认的最小陈述；
- 学术争议；
- 不应作出的推断；
- 若进入设定层，原创从哪一处空白开始。

被重新发现的材料可以让条目离开静默长廊，但旧版本与修订理由必须保留。

## 已保留名字

- [特内姆与特内梅特](./egyptian/tenem-and-tenemet.md)：《棺材文》76 中与“无方向／游移”相关的原初一对，后续八神组合常将其替换。
- [肯提阿门提乌](./egyptian/khentyamentiu.md)：阿拜多斯早期“西方人之首”，其地方位置后来被奥西里斯吸收。

两页都拒绝为证据空白生成现代性格。静默长廊保存的不是“冷门设定素材”，而是来源尚不足以支撑完整神座的名字。
