## 文明档案记录什么

- 使用的语言与核心文本；
- 主要历史时期；
- 神系是否真的存在统一正典；
- 地方祭祀与书面神话的关系；
- 后世宗教、国家和文学造成的改造；
- 与其他文明真实发生接触的道路。

跨文明接触的通用规则见 [《文明通道》](./corridors.md)。
