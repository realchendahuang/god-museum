中国馆当前以甲骨、商周礼器、《尚书》《诗经》《山海经》《楚辞》《淮南子》《史记》等材料建立帝、天命、社稷、神话地理、战争、洪水和秦汉国家祭祀网络；又以《三国志》《老子想尔注》、六朝灵宝经系、《真灵位业图》《宋史·礼志》等建立天师道、三清与玉皇的历史形成层。佛教传播、地方神与文学面相仍按年代接入，不把“三清—玉皇—诸部—地方神”误写为自上古不变的统一神谱。

## 文明档案记录什么

- 使用的语言与核心文本；
- 主要历史时期；
- 神系是否真的存在统一正典；
- 地方祭祀与书面神话的关系；
- 后世宗教、国家和文学造成的改造；
- 与其他文明真实发生接触的道路。

跨文明接触的通用规则见 [《文明通道》](./corridors.md)。
