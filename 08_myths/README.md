# 神话剧场

神话剧场保存事件，而不是把事件拆散到单个神明的简介里。

## 事件范围

- 创世与天地重组；
- 神战、继承与王权更替；
- 英雄远征与试炼；
- 洪水、瘟疫、饥荒与灾变；
- 下降冥界、复生与禁忌破坏；
- 末日、轮回与新世界。

## 版本规则

同一事件在不同文本中的版本并排保存。跨文明相似事件默认建立 motif-analogue，不宣布为同一场历史。

一份事件档案至少包含：

- 稳定 ID 与事件名称；
- 所属传统；
- 参与神座、英雄、异兽、圣物和地点；
- 各文本版本；
- 版本之间的主要分歧；
- 后世重述；
- God-Museum 原创延伸。

神话事件使用 myth.传统.名称 作为稳定 ID。
