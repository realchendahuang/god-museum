# 太阳夜航

> 日落之后，太阳不是从世界设定中消失：它穿行杜阿特十二时，与奥西里斯的死亡力量结合，抵抗阻断更新的阿佩普，并在东方重新诞生。

## 事件档案

| 项目 | 内容 |
| --- | --- |
| 核心语料 | 《阿姆杜阿特》及其他冥界诸书；亡灵书中的相关太阳咒语与图像 |
| 核心行动者 | [拉](../../02_deities/egyptian/ra.md)、[奥西里斯](../../02_deities/egyptian/osiris.md)、太阳船众神与守卫 |
| 主要敌手 | [阿佩普](../../06_beasts/egyptian/apep.md)及其他阻断、惩罚与危险存在 |
| 主要空间 | [杜阿特](../../07_places/egyptian/duat.md)，按时辰、门、洞窟、水域与尸体等多种方式组织 |
| 事件结果 | 太阳获得更新，清晨以新的形态出现；死者也争取加入这一循环 |

## 原典层：十二时不是十二关游戏

### 进入西方

太阳在西方地平线进入杜阿特。航行既是空间移动，也是时间按夜间时辰展开。沿途存在知名或匿名神、受祝福死者、守门者、蛇形力量与受罚者；知道名字和秩序能够让通行发生。

### 与奥西里斯相接

夜间深处，拉的巴与奥西里斯的尸体／死亡力量相接。两神没有永久融合成一个无差别神座：流动的太阳需要死亡中的再生潜能，奥西里斯也进入太阳更新的宇宙循环。

这一结构让“死亡”和“生命”不再是一次切换，而是每天必须重新建立的关系。

### 对抗阿佩普

阿佩普试图搁浅、吞噬或中断太阳船。咒语、束缚、切割、焚烧和太阳船护卫共同压制它。胜利不是把混沌永久杀死；夜航次日仍会再次发生。

塞特在部分太阳船传统中以危险强力刺击阿佩普，说明秩序可调用不稳定力量，但不能据此把塞特的奥西里斯冲突删除。

### 东方再生

太阳在黎明获得更新形态，常以凯布利及圣甲虫图像表达“成为／生成”。拉、凯布利、阿图姆是太阳时间与创生状态中的关系，不应简单列成“早中晚三个人格皮肤”。

## 语料版本

| 语料 | 主要差异 |
| --- | --- |
| 《阿姆杜阿特》 | 以十二时组织太阳船、神灵、地貌和更新；早期与新王国王墓关系紧密 |
| 《门之书》 | 以门与守卫组织时序，包含不同群体和审判图景 |
| 《洞窟之书》《地之书》 | 更强调洞窟、尸体、惩罚和太阳再生的其他构图 |
| 《亡灵书》 | 死者通过选用咒语、名字和图像争取白昼出行、加入太阳船或抵御危险 |

这些作品互相交流母题，却不是一部“太阳夜航系列”的固定章节顺序。

## 馆藏关系

- [阿佩普](../../06_beasts/egyptian/apep.md)：永远试图阻断，而不是一次性被击杀的关底怪物；
- [奥西里斯](../../02_deities/egyptian/osiris.md)：死亡与复苏力量进入每日循环；
- [安卡](../../05_relics/egyptian/ankh.md)：生命符号可在太阳与神向王、死者给予生命的图像中出现，但不是太阳船发动装置；
- [法老](../../04_heroes/egyptian/pharaoh.md)：王室墓葬最早大规模采用冥界诸书，后来材料使用范围变化。

## 设定层

太阳夜航是古埃及诸神域与众神殿之间唯一每天关闭又重开的通道。通道关闭时，共殿不会陷入普遍黑夜；只有埃及档案进入杜阿特的时间层。

其他文明的太阳可以在第三殿观察这套更新，却不能搭乘太阳船自动取得埃及神名。比较只在日落、夜间危险和再生母题上建立 motif-analogue。
