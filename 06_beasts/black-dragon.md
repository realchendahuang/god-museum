# 黑龙

> 当前可确认的黑龙只存在于《淮南子》补天段落中的一个短动作：“杀黑龙以济冀州”；名字、形态、来历和独立故事都不能从这七个字之外任意补写。

## 现存证据

《淮南子·览冥训》在女娲炼石补天、断鳌足立四极的连续行动中写到“杀黑龙以济冀州”，随后又以芦灰止水。黑龙因此明确参与灾后复序事件，并与冀州得到救济有关。

文本没有在同一段说明：

- 黑龙是否造成“四极废，九州裂”；
- 它是否等同于后世某条有名神龙；
- 它的血缘、居所、体型与完整职能；
- “黑”指颜色、方位、五行象征还是其他分类。

## 最小档案

| 项目 | 当前结论 |
| --- | --- |
| 名称 | 黑龙 / 黑龍 |
| 事件 | 女娲补天与灾后复序 |
| 行动 | 被女娲杀死，以济冀州 |
| 置信状态 | 名称与动作确认；其余 fragmentary |
| 禁止合并 | 不与共工、相柳、应龙或其他龙类自动视为同一身份 |

## 整理层

黑龙进入异兽庭院的意义，不是提供又一只可供战斗的怪物，而是展示档案如何诚实面对极短材料。它与 [弗栗多](./vedic/vritra.md) 都能被放进“水患中的蛇龙敌对者”母题对照，但双方文本、行动者和灾变结构完全不同。

## 设定层

该馆室仍处于 proposed 状态。众神殿只保留一条黑色轮廓和“济冀州”四字，不为它生成眼睛、鳞片或声音。

未来如果发现并纳入新的文本、注释或图像，馆室会增加面相；在此之前，任何完整人格都只能作为明确标注的原创实验，不能进入正式设定正典。
