# 异兽庭院

异兽庭院保存神兽、怪物、恶魔、精灵和不能稳定归入神或人的非人存在。

档案必须先回答它在本土传统中是什么，再决定如何翻译。monster、demon、dragon、spirit 等现代通称不能替代原词。

每个条目应记录：

- 原文名称与词义；
- 所属传统、地区和时期；
- 形态与行为；
- 与神、英雄和地点的关系；
- 是否存在善恶、神兽/怪物等后世重分类；
- 图像与文本之间的差异。

异兽条目使用 being.传统.名称 作为稳定 ID。

## 已开放馆室

- [弗栗多](./vritra.md)：在吠陀赞歌中阻挡水流、与因陀罗及金刚杵形成完整敌对事件。
- [黑龙](./black-dragon.md)：《淮南子》补天段落中仅存短暂动作的 fragmentary 条目。

这两个条目故意展示异兽档案的两个极端：弗栗多拥有可连续阅读的战斗结构，黑龙则必须让未知保持可见。内容多少不决定存在价值，证据边界决定我们能写到哪里。
