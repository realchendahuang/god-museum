# 异兽庭院

异兽庭院保存神兽、怪物、恶魔、精灵和不能稳定归入神或人的非人存在。

档案必须先回答它在本土传统中是什么，再决定如何翻译。monster、demon、dragon、spirit 等现代通称不能替代原词。

每个条目应记录：

- 原文名称与词义；
- 所属传统、地区和时期；
- 形态与行为；
- 与神、英雄和地点的关系；
- 是否存在善恶、神兽/怪物等后世重分类；
- 图像与文本之间的差异。

异兽条目使用 being.传统.名称 作为稳定 ID。

## 已开放馆室

- [弗栗多](./vedic/vritra.md)：在吠陀赞歌中阻挡水流、与因陀罗及金刚杵形成完整敌对事件。
- [黑龙](./black-dragon.md)：《淮南子》补天段落中仅存短暂动作的 fragmentary 条目。
- [精卫](./chinese/jingwei.md)：炎帝少女女娃溺亡后的鸟形身份，持续填海但没有被原典保证成功。
- [应龙](./chinese/yinglong.md)：蓄水攻蚩尤、杀敌、失去升天能力并成为求雨模仿对象的复杂龙形存在。
- [相柳氏](./chinese/xiangliu.md)：共工九首蛇身之臣，被杀后的血仍使土地无法种植五谷。
- [烛龙／烛九阴](./chinese/zhulong.md)：以开闭眼组织明暗、与风雨相应的宇宙身体，不是普通火龙。
- [阿佩普](./egyptian/apep.md)：反复阻断太阳夜航与宇宙更新的原型敌手，不能简单归为普通神或妖魔。
- [阿米特](./egyptian/ammit.md)：称心审判中权能严格受场景限制的吞噬者，不是冥界统治者或战兽。
- [安祖](./mesopotamian/anzu.md)：从狮首鹰保护图像到夺取命运泥板的史诗叛乱者，保留媒介差异。
- [mušḫuššu](./mesopotamian/mushhushu.md)：可追踪神徽传承的复合蛇龙，不与提亚马特或恐龙混同。
- [胡姆巴巴](./mesopotamian/humbaba.md)：雪松林受命守护者与英雄敌手的双重身份。
- [天牛](./mesopotamian/bull-of-heaven.md)：由神权释放的城市灾难，也是恩奇都死亡判决的直接前因。
- [洛坦](./ugaritic/lotan.md)：逃窜、扭曲的七首蛇敌；与雅姆、Leviathan 和屠龙母题相关但不直接同一。

这些条目展示异兽分类的多种边界：精卫由人死后变形，应龙在战争与仪式之间移动，相柳留下生态污染，烛龙的身体直接参与宇宙运行；黑龙则必须让未知保持可见。内容多少不决定存在价值，证据边界决定我们能写到哪里。
